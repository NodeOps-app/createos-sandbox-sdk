createos-sandbox SDK

The createos-sandbox-sdk is a TypeScript SDK for the createos-sandbox VM sandbox control plane.

Key Features

  • Zero runtime dependencies
  • ESM-only, built with tsc
  • Typed HTTP client with retry logic
  • Full sandbox lifecycle management

Quick Start

Install the SDK and create your first sandbox in seconds:

import { Sandbox } from "createos-sandbox-sdk";
const sb = await Sandbox.create({ shape: "s-1vcpu-1gb", rootfs: "devbox:1" });
await sb.destroy();

Primitives

Primitive Description
createSandbox Boot a new VM
runCommand Run a buffered command
files.upload Upload a file to the guest
files.download Download a file from the guest
destroy Tear down the sandbox
